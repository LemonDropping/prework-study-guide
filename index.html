HTML

  • Use opening and closing angle brackets.
  • Head element contains info about our webpage.
  • Body element represents the visible content shown and seen by the user.
  • DOCTYPE html establishes html type.
  • Elements inside other elements are called a child element.
  • Parent elements have children in them.
  • Semantic elements describe its content and roll on a webpage.

Git

  • Git is version control
  • We connect our local cpu to our github repository.
  • Git branches allow the user to have others look at the code and not work on the main code.
  • Pull request allows team members to review your code.
  • Merge and deploy merges the created branch after content has been approved.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
