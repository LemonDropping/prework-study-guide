Prework Study Guide

HTML

  • Use opening and closing angle brackets.
  • Head element contains info about our webpage.
  • Body element represents the visible content shown and seen by the user.
  • DOCTYPE html establishes html type.
  • Elements inside other elements are called a child element.
  • Parent elements have children in them.
  • Semantic elements describe its content and roll on a webpage.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS is our styling code.
  • 3 ways to style a webpage using Inline CSS, Internal CSS style sheet, external CSS style sheet.
  • Most common way to style is using an external style sheet.
  • Declarations contain CSS property, ex: color:blue, color is the property.
  • style embeds entire style sheet directly to HTML file

Git

  • Git is version control
  • We connect our local cpu to our github repository.
  • Git branches allow the user to have others look at the code and not work on the main code.
  • Pull request allows team members to review your code.
  • Merge and deploy merges the created branch after content has been approved.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • The strict equality operator (===) checks to see if two values are equal, and returns a Booean result true.
  • Group of related data is an array.
  • Functions are made with () parentheses.
  • All strings use quotation marks.
